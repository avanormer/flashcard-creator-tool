What I'm building: 
I am building a flashcard maker/rapid flashcard quiz simulation that allows people to create and study their own flashcards in order to learn through repetitive memorization online.

Who it's for / Why: (who would use this? what problem does it solve?)
This will be created as a resource for students who are looking for ways to memorize course content such as key concepts, vocab terms, formulas, and memory based applications. This is created to solve the problem of wasting plenty of index cards and paper while still having a reliable, digital study tool that can be utilized anywhere.


Core features: (3-5 things it will do)
- It will be able to create flashcards with a question on the front and answer on the black
- Flashcard will be able to be flipped to reveal the answer by click
- Flashcards will be able to be sorted based off of memorized, viewed, need more time studied on, etc...


What I don't know yet: (what JavaScript concepts will you need to figure out?)
- DOM (how to update) with new cards
- how switching between new and old cards will be stored or managed
- how to implement fliping card visual
