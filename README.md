# What this tool does:

This tool is a flashcard maker that allows people to create and study their own flashcards and then be quized on the content they put on the flashcards. This tool will fulfill multiple uses; such as creating flashcards with a nquestion on the front and answer on the black, flashcards to be flipped to reveal the answer by click, and flashcards be able to be sorted based off of memorized, viewed, need more time studied on, etc...

## How to Use

1. **Open the app**: Load `index.html` in your web browser
2. **Add a flashcard**: Enter a question and answer in the form, then click "Add Flashcard"
3. **Study**: Use the "Flip Card" button to reveal answers
4. **Navigate**: Click "Next Card" or "Previous Card" to move through your deck
5. **Track progress**: Mark cards as "Memorized" or "Needs Practice" as you study
6. **Reset**: Click "Reset All Cards" to clear the deck and start over
